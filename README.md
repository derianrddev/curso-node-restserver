# REST API 

REST API with Node.js, Express and MongoDB of a grocery store with a CRUD of users, categories and products, user authentication with email and password and google and searches.

### Installation

First, clone the repo

```sh
git clone https://github.com/derianrddev/curso-node-restserver.git
```

move to the folder

```sh
cd curso-node-restserver
```

install dependencies:

```sh
npm install
```

execute the project

```sh
npm run dev
```
